The local published-paper folder `/Users/dlev2617/Documents/Papers/~05-Published/IJTM-PerspectivesOnEfficiency` contains drafts, letters, presentation material, and one compact data/model workbook: `Accessibility.xls`. Only the workbook was staged. Drafts, correspondence, comments, presentations, and public reference PDFs were excluded.

The broader ramp-meter data tree at `/Users/dlev2617/Documents/Data/~Nexus_Data/Ramps` was checked as shared context. It contains raw and derived ramp-meter shutdown files used by related papers, but those files are not duplicated here. This package keeps the paper-specific workbook and documents the shared context for future deduplication.
